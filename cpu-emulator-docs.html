CPU Emulator + Assembler Documentation

Quick Start

  1. Open CPU EMULATOR from the GAMES menu.
  2. Write assembly in the ASSEMBLY SOURCE panel.
  3. Click ASSEMBLE + LOAD.
  4. Use STEP or RUN to execute from 0x0500.

Assembly Syntax

  • Registers: R0 to RF.
  • Immediates: decimal (42) or hex (0x002A).
  • Comments: start with ; or #.
  • Labels: LOOP: on their own line or before an instruction.
  • Instruction formats:
    • OP
    • OP Rn
    • OP Rn, Rm
    • OP Rn, imm
    • OP imm

Example Program

START:
  LFI R1, 0x0005
  LFI R2, 0x0003
  ADD R1, R2
  HALT
